# ペイント リポジトリ
開発中。

## ディレクトリ
リポジトリの主なディレクトリを以下に示します。

| ディレクトリ | プロジェクト | 備考 |
| ------- | --- | --- |
| build   | --- | ビルドした実行可能ファイルを格納します。 |
| debug   | --- | デバッグ オプションを付けてビルドした実行可能ファイルを格納します。 |
| docs    | --- | GitHub Pages 用 HTML ファイルを格納します。 |
| draw    | 描画 | 描画アプリケーションのソース ファイルを格納します。 |
| icons   | --- | 各アプリケーションに埋め込む画像を格納します。 |
| locale  | --- | 生成したメッセージ カタログを格納します。 |
| msg     | --- | メッセージ テーブルからメッセージ カタログを作成する方法を提供します。 |
| paint   | ペイント | ペイント アプリケーションのソース ファイルを格納します。 |
| share   | --- | 各アプリケーションに共通する処理を記述したソース ファイルを格納します。 |
| release | --- | 最適化オプションを付けてビルドした実行可能ファイルを格納します。 |
| viewer  | ピクチャ ビューアー | ピクチャ ビューアー アプリケーションのソース ファイルを格納します。 |

## メイクファイル
メイクファイルの主なコマンドを以下に示します。

| コマンド | 備考 |
| -------------- | --- |
| make           | 各アプリケーションをビルドします。 |
| make clean     | 生成したディレクトリを削除します。 |
| make debug     | デバッグ オプションを付けて各アプリケーションをビルドします。 |
| make release   | 最適化オプションを付けて各アプリケーションをビルドします。 |
| make uninstall | インストールした GSettings スキーマ ファイルを削除します。 |

## コンパイラー
### リソース コンパイラー
`glib-compile-resources` はリソース仕様をコンパイルしてリソース ファイルを生成します。
リソース仕様ファイルの拡張子は `.gresource.xml` です。
リソース ファイルの拡張子は `.gresource` です。

### スキーマ コンパイラー
`glib-compile-schemas` はすべての GSettings スキーマ ファイルをコンパイルしてスキーマ キャッシュを生成します。
スキーマ ファイルの拡張子は `.gschema.xml` です。
キャッシュ ファイルの名前は `gschemas.compiled` です。
